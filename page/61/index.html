box-shadow使用

发表于 2018-07-30 更新于 2020-03-31 分类于 css 阅读次数：

box-shadow 前言

box-shadow 是 css3 的一个新属性，用来实现阴影效果，阴影分为内阴影和外阴影两个效果，可以通过逗号添加多个阴影效果。

box-shadow 阴影用法

box-shadow 参数解释

@1：inset

有 inset 则为内阴影，没有 insert 则为外阴影，默认为外阴影。

@2：offset-x

横向阴影的大小，正值阴影在右边，负值阴影在左边，0 的时候阴影中盒子后面，看不见的，如果有 blur-radius 值会有模糊效果。

@3：offset-y

纵向阴影的大小，值同 offset-x。

@4：blur-radius

阴影的模糊程度，值越大，阴影越模糊。

@5：spread-radius

阴影的扩大缩小，正值时，阴影扩大；负值时，阴影缩小，默认为 0，和盒子同样大。
