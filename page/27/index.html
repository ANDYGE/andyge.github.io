zabbix之IMPI监控

发表于 2019-08-30 更新于 2020-03-31 分类于 zabbix 阅读次数：

要点

IPMI：智能平台管理接口。

开放的免费标准，监控硬件、管理硬件、物理状态等，比如：温度、电压、电风扇、电源，独立于操作系统存在，无论关机还是开机，只要接通电源，即可监控。

数据监控流程

zabbix-server–>IPMI Poller–>Device With IPMI–>IPMI Poller–>zabbix-server

Zabbix-server 配置

配置 DELL 服务器 IPMI 相关设置

登录 iDrac 管理界面

  • iDrac 设置–》网络–》IPMI 设置–》启用
  • iDrac 设置–》用户验证–》点击一个已禁用的用户 id，选择用户配置–》启用–》用户名：zabbix–》密码：zabbix–》IPMI 用户权限–》用户–》应用

在 zabbix-server 登录验证

在 zabbix-web 配置监控项

  • 添加 ipmi 主机，添加 IPMI 接口
  • 通过主机的IPMI页签，添加算法，权限级别，及登录账号
  • 添加监控项，类别选择 IPMI，keys 自定义，Sensor 选择刚才的Fan1
  • 回到监控面板，选择最新数据查看

SNMP的安装配置

发表于 2019-08-29 更新于 2020-03-31 分类于 zabbix 阅读次数：

SNMP

简单网络管理协议（SNMP） 是专门设计用于在 IP 网络管理网络节点（服务器、工作站、路由器、交换机及 HUBS 等）的一种标准协议，它是一种应用层协议。 SNMP 使网络管理员能够管理网络效能，发现并解决网络问题以及规划网络增长。通过 SNMP 接收随机消息（及事件报告）网络管理系统获知网络出现问题。

SNMP 是管理进程（NMS）和代理进程（Agent）之间的通信协议。它规定了在网络环境中对设备进行监视和管理的标准化管理框架、通信的公共语言、相应的安全和访问控制机制。网络管理员使用 SNMP 功能可以查询设备信息、修改设备的参数值、监控设备状态、自动发现网络故障、生成报告等。
