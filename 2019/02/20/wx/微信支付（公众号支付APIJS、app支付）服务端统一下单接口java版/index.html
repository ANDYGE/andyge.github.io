微信支付（公众号支付APIJS、app支付）服务端统一下单接口java版

发表于 2019-02-20 更新于 2020-03-31 分类于 wx 阅读次数：

流程图

  1. 预支付,后台系统跟微信后台交互,给小程序返回参数 统一下单
  2. 小程序拿到参数进行预支付
  3. 用户确认支付,//跟微信后台交互,这一步我们不需要管
  4. 微信后台回调我们的后台系统的接口, 处理我们自己的业务逻辑 回调处理支付结果

微信公众号支付 APIJS：

要完整的实现微信支付功能，需要前后端一起实现，还需要微信商户平台的配置。这里只是涉及服务端的代码。
