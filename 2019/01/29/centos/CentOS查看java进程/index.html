查看 Java 进程获取 pid 号

显示出所有的 java 进程，移除掉当前的 grep 进程

ps aux

最常用的方法是 ps aux,然后再通过管道使用 grep 命令过滤查找特定的进程,然后再对特定的进程进行操作。

ps -ef 与 ps aux

-ef 是 System V 展示风格，而 aux 是 BSD 风格。

如果想查看进程的 CPU 占用率和内存占用率，可以使用 aux;

如果想查看进程的父进程 ID 和完整的 COMMAND 命令，可以使用 ef;
