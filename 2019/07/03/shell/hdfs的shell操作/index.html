hdfs的shell操作

发表于 2019-07-03 更新于 2020-03-31 分类于 hadoop 阅读次数：

启动 Hadoop 集群

-help：查看命令

-ls：显示目录信息

-mkdir：创建目录

-moveFromLocal：从本地剪切粘贴到 HDFS

-appendToFile：追加一个文件到已经存在文件的后面

-chgrp、-chmod、-chown：与 Linux 文件系统用法一致，修改文件所属权限

-copyFromLocal：从本地拷贝文件到 HDFS

-copyToLocal：从 HDFS 拷贝到本地

-cp：从 HDFS 的一个路径拷贝到另一个路径

-mv：在 HDFS 上移动文件

-get：等同于 copyToLocal，就是从 HDFS 下载文件到本地

-getmerge：合并瞎子啊多个文件，比如 HDFS 的目录/aaa/下有多个文件：log.1,log.2,log.3

-put：等同于 copyFromLocal

-tail：显示一个的末尾

-rm：删除文件

-rmdir：删除空目录

-du：统计文件夹的大小信息

-setrep：设置 HDFS 中文件的副本数量
