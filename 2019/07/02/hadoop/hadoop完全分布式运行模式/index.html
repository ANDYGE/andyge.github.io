rsync 远程同步工具

rsync 主要用于备份和镜像，具有速度快、避免复制相同内容和支持符号链接的优点。

rsync 和 scp 区别：用 rsync 做文件复制要比 scp 的速度快，rsync 只对差异文件做更新。scp 是把所有文件都复制过去。
