站点根目录

例如在微信开发中经常要进行文件校验，将某个指定的文件放置到站点根目录或者某个目录下，例如微信要访问http://owl.weiqiaon.cn/322NXaSqwJ.txt，应该放置在 SpringBoot 项目的什么地方呢？

答案是 src/main/resoureces/static 目录即可。

spring boot 加载资源路径配置

spring boot 默认加载文件的路径：

  • /META-INF/resources/

SpringBoot-读取 classpath 下文件

开发过程中，必不可少的需要读取文件，对于打包方式的不同，还会存在一些坑，比如以 jar 包方式部署时，文件都存在于 jar 包中，某些读取方式在开发工程中都可行，但是打包后，由于文件被保存在 jar 中，会导致读取失败。这时就需要通过类加载器读取文件，类加载器可以读取 jar 包中的 class 类当然也可以读取 jar 包中的文件。
