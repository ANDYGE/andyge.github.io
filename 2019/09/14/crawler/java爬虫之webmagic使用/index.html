java爬虫之webmagic使用

发表于 2019-09-14 更新于 2020-03-31 分类于 crawler 阅读次数：

要点

WebMagic 是一个简单灵活的 Java 爬虫框架。基于 WebMagic，你可以快速开发出一个高效、易维护的爬虫。

官网

http://webmagic.io/

特性

  • 简单的 API，可快速上手
  • 模块化的结构，可轻松扩展
  • 提供多线程和分布式支持
