PWA

PWA 是 Progressive Web App 的英文缩写， 翻译过来就是渐进式增强 WEB 应用， 是 Google 在 2016 年提出的概念，2017 年落地的 web 技术。目的就是在移动端利用提供的标准化框架，在网页应用中实现和原生应用相近的用户体验的渐进式网页应用。

  • 官网：https://developers.google.com/web/progressive-web-apps/
  • 介绍：https://www.jianshu.com/p/098af61bbe04

UmiJS

umi，中文可发音为乌米，是一个可插拔的企业级 react 应用框架。

dva
