wepy-redux使用笔记

发表于 2018-07-03 更新于 2020-03-31 分类于 wechat 阅读次数：

概述

可以在小程序里使用 Redux 进行状态管理， Redux 是一个前端状态管理的容器，对于构建大型应用，对里面共享数据、状态的管理非常方便，学过 React 的同学对它应该不陌生。

wepy 框架本身是支持 Redux 的，我们在通过wepy init standard project-name构建项目的时候，将询问是否安装 Redux 选择 y 就好了，会自动安装依赖并添加样例代码。

要点

1、 能够简单的实现状态数据绑定和事件动作绑定。
2、 有两个重要的概念state和action，state中包含的是状态数据，action是触发状态数据变化的动作。
