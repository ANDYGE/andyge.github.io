2.2. 生成目录

在 vs code 插件中搜索markdown toc，安装，即可快速生成目录。注意是星星数最多的那个。作者是AlanWalk。

利用该插件不需要显式录入节点序号，该插件可以自动生成。注意该插件要求文章中包含有#(h1),##(h2)…，才能生成从 1 开始的节点号。

3. 填坑系列

3.1. 转义

有些特殊字符不建议在文本中使用，可以使用转义字符，例如

@Convert(converter = EnumConvert<StatusEnum>.class)

3.2. html 表格

用户 HTML 在 markdown 中绘制表格时，预览时会生成很多空行，这是由于 HTML 中的回车符造成的，解决的办法就是把这个 html 用压缩工具压缩一下就可了。
