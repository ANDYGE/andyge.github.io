4. 技巧

4.1. 填坑

4.1.1. 特殊字符需转码

如果正文中包含 if=”{{!foo}}”， 在预览或者发布的时候，报如下错误：unexpected token: }}，这是因为在 hexo 中双括号会被解析，双括号里的内容会被当做变量解析，解决办法就是使用转义符：if="&#123;&#123;!foo&#125;&#125;"，这种方式时，`根本不会把转义过后的内容反转义回去。

4.1.2. 域名映射

  • 在域名“解析”添加 cname 类型的记录
  • 在站点的根目录下创建 CNAME 文件，内容为该域名，如:swiftist.cn
  • 同时注意 CNZZ，畅言，leanclound 的域名同步更改
