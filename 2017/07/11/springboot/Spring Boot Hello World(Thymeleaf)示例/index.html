发表于 2017-07-11 更新于 2020-03-31 分类于 java 阅读次数：

这是一个 Spring Boot Web 应用示例，使用嵌入式 Tomcat + Thymeleaf 模板引擎，并将其作为可执行 JAR 文件。

使用的相关技术：

  1. Spring Boot 1.4.2.RELEASE
  2. Spring 4.3.4.RELEASE
  3. Thymeleaf 2.1.5.RELEASE https://www.cnblogs.com/ityouknow/p/5833560.html
  4. Tomcat Embed 8.5.6
  5. Maven 3
  6. Java 8
